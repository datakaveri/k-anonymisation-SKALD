SKALD Pipeline — Error Code Reference
========================================

HTTP | Code                        | Error Message                                              | Suggested Fix
-----|-----------------------------|------------------------------------------------------------|--------------------------------------------------------------------------------
400  | CONFIG_NOT_FOUND            | No JSON config file found in config/                       | Create a config/ directory and add a JSON configuration file. Refer to the schema reference for required fields.
400  | CONFIG_PARSE_ERROR          | Failed to parse JSON configuration                         | Validate your config JSON with a linter. Ensure all brackets and quotes are balanced.
400  | CONFIG_MISSING_FIELD        | Config file is missing a required field                    | Required fields: data_type, quasi_identifiers, k_anonymize, output_path.
400  | CONFIG_INVALID              | Config field value is out of range                         | k must be >= 1, suppression_limit must be 0.0-1.0, size factors must be positive integers.
400  | CONFIG_INVALID_VALUE        | Config field value is out of range                         | k must be >= 1, suppression_limit must be 0.0-1.0, size factors must be positive integers.
422  | DATA_DIR_MISSING            | data/ directory does not exist                             | Create a data/ directory and place exactly one CSV file inside it.
422  | DATA_NO_CSV                 | No CSV file found in data/                                 | Add exactly one CSV file with a header row to data/.
422  | DATA_EMPTY                  | CSV file exists but is empty                               | Ensure the file has a header row and at least one data row.
422  | DATA_COLUMN_MISSING         | A quasi-identifier column was not found in the CSV header  | Column names are case-sensitive — verify they match the config exactly.
422  | PREPROCESS_COLUMN_MISSING   | A preprocessing target column was not found in CSV header  | Check suppress, masking, hashing, tokenization, fpe, encrypt, and charcloak column names against the actual CSV header.
422  | PREPROCESS_CONFIG_INVALID   | A preprocessing config entry is malformed                  | Each entry must be an object with a 'column' field and valid parameters.
422  | ANON_INFEASIBLE             | k-anonymity cannot be satisfied with current settings      | Try: (1) increase suppression_limit (e.g. to 0.05), (2) decrease k, (3) add more records, (4) increase size factors for numerical QIs.
422  | ANON_NO_QIS                 | No quasi-identifiers are defined                           | Add at least one entry to quasi_identifiers.numerical or quasi_identifiers.categorical in the config.
422  | ENCODING_FAILED             | A data encoding error occurred                             | Check that numerical QI columns contain only numeric values and no blanks.
500  | IO_READ_FAILED              | A required file could not be read                          | Check that the file exists and the process has read permissions.
500  | IO_WRITE_FAILED             | A file could not be written                                | Check disk space and write permissions for the output/ and chunks/ directories.
500  | IO_PERMISSION_DENIED        | Permission denied on a file or directory                   | Run the pipeline with appropriate filesystem permissions.
500  | INTERNAL_ERROR              | An unexpected internal error occurred                      | Check output/pipeline.log for a full trace and contact support.


HTTP Grouping
-------------
400  Bad Request   — Config is wrong or missing (caller error)
422  Unprocessable — Data cannot be processed (data/anonymization issue)
500  Server Error  — Disk, permissions, or unexpected crash
